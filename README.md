# vue-tareas ⚡

## Iniciar proyecto 
```
npm install
```

### Compilación y recarga del desarrollo
```
npm run serve
```

### Compila y minifica para la producción
```
npm run build
```

### Fichas y correcciones
```
npm run lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
